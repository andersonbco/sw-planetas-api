# sw-planetas-api

**- Pré-requisitos**
- Maven 3.5.4
- Java 8

**- Feito com:**
- Spring Boot 2.0
- Mongo 3.6.3

**- Para subir a aplicação:**
- Subir o Mongo na porta default (27017)
- Na raiz do projeto executar o comando `mvn spring-boot:run`
- O swagger da aplicação ficará disponível em `localhost:8080/swagger-ui.html`
